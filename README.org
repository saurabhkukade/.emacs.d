 My Personal emacs configuration.

* Prerequisites
 - *GNU Emacs Version >= 25.1.1*
 - *Python 2.7.9*
 - *Pip
 - jedi, virtualenv, epc, pylint (pip install)

** Linux

Make sure that you are connected to internet while bootstrapping.
